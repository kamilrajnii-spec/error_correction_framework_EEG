# Optimizing Error Correction Frameworks for EEG Signal Processing

## 1. Project Overview
This thesis proposes a validated hybrid approach using **Discrete Wavelet Transforms (DWT)** and **1D-Convolutional Neural Networks** to bridge the "Deployment Gap" in real-time neuro-diagnostics. This implementation is optimized for Edge-AI deployment on commodity hardware (Ryzen 5 3500U).
